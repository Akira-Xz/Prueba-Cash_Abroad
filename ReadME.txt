Para hacer que este codigo pueda correr es necesario descomprimirlo en una carpeta propia, una vez alli, se podra abrir utilizando VS Code, las librerias de node y express ya estan importadas, pero sin embargo para asegurar su funcionamiento se recomienda volver a instalarlas en el directorio raiz de a carpeta, posteriormente correr el codigo utilizando *node app.js* , cuando salga el mensaje de que el programa esta corriendo entonces se usara algun programa como Postman o ThunderClient en el que se copiara la url del puerto y los endpoints que se han creado en el programa, haciendo una peticion post y mandandola en formato JSON con la moneda que se quiere convertir, la respuesta se mandara en la consola del programa de peticiones usado
